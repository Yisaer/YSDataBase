# YSDataBase
A simple Key-Value DataBase by Python


## 使用方法
	
	## 设置/更新 键值
	
	python -m YSDB.Tool Database.db set Key Value 

	## 查询键值
	
	python -m YSDB.Tool Database.db get Key
	
	Value
	
	## 删除键值
	
	python -m YSDB.Tool Database.db delete Key
